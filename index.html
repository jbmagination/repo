Dress 1.0 (DRM-Free)

Due to how buggy the DRM for Dress is, I decided to take it upon myself to disable the DRM entirely. The tweak is open source, so I forked it and began working on it.

You can find Dress 1.0 (DRM-Free)'s source code here.

This has now been removed from the repo by request of the original owner, Litten. The tweak will still show up, but it will not be downloadable. Additionally, the GitHub repository for the Dress repo has also been removed. Please update to 1.1, which fixes the DRM issue by removing it entirely.
